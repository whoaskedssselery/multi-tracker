# App Icons

Place the following PNG files here for PWA support:

- `icon-192.png`   — 192×192 px  (Android/Chrome home screen)
- `icon-512.png`   — 512×512 px  (splash screen)
- `icon-maskable.png` — 512×512 px, with safe zone padding (Android adaptive icon)
- `icon-180.png`   — 180×180 px  (Apple Touch Icon — "Add to Home Screen" on iPhone)

Recommended: use the same sage-green brand gradient (#6B8F71 → #4A6E51)
with a white "M" letter centered on the icon.
